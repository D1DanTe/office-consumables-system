# Система учёта расходных материалов

Веб-приложение для учёта офисных расходников.

## Возможности

- Авторизация пользователей
- Роли пользователей
- Управление товарами
- Заявки сотрудников
- Контроль остатков
- Поступления товаров
- Поставщики
- Автоматическое списание при выдаче
- Контроль минимальных остатков

## Технологии

- Python 3
- Flask
- SQLAlchemy
- SQLite
- Flask-Login

## Установка

pip install -r requirements.txt

python app.py
